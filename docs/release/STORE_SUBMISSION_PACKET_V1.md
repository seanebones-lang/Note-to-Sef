`Note to Self is a local-first note app built for quick capture and clean organization.

Capture ideas in seconds
- Write or dictate notes when thoughts happen.
- Keep everything in one simple inbox.

Organize without friction
- Sort notes into categories and tags.
- Pin important notes and archive old ones.

Find anything fast
