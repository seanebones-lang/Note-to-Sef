## GitHub Pages Setup (No Actions Fallback)

If GitHub Actions Pages deployment is blocked, publish directly from branch:

1. Open `https://github.com/seanebones-lang/Note-to-Sef/settings/pages`
2. Build and deployment -> Source: `Deploy from a branch`
3. Branch: `main`
4. Folder: `/docs`
